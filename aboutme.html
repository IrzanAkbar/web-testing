Tentang Saya

Hai, saya Andi Irzan Akbar H., seorang pelajar yang penuh semangat dan berdedikasi dari Wajo, Sulawesi Selatan. Saya sedang menikmati masa remaja dengan semangat belajar dan mengeksplorasi dunia baru.

Minat dan Kegiatan

Di luar kegiatan sekolah, saya gemar bermain game, yang membantu saya mengembangkan kemampuan problem-solving dan berpikir kritis. Selain itu, saya juga sangat tertarik dengan dunia pemrograman, terutama pengembangan web. Kode dan algoritma adalah tantangan yang menyenangkan untuk dipecahkan!

Keahlian

  • Pemrograman Web (HTML, CSS, JavaScript)
  • Pengembangan Bot WhatsApp (NodeJs)
  • Server Game GTPS (C++) [Sedang Belajar]

Pengalaman

Saya memiliki pengalaman dalam mengembangkan bot WhatsApp yang berbasis Node.js, di mana saya mengintegrasikan fitur-fitur pengenalan suara dan respons otomatis. Saya juga telah membuat server game GTZL dan ZLPS menggunakan C++, yang melibatkan pengelolaan database, algoritma, dan optimasi server.

Tujuan

Saya bersemangat untuk terus belajar dan mengembangkan keahlian saya dalam pengembangan web. Saya ingin menciptakan aplikasi web yang inovatif dan bermanfaat bagi orang lain.
